let, const, var 차이

1. scope let, const는 block 스코프를 갖고, var는 function 스코프를 갖는다
2. var 키워드는 같은 이름의 변수명을 사용해서 재선언이 가능하지만, let, const키워드로 선언할경우 재 선언은 불가능하다.


var test = 'test';
var test = 'test2';

let test2 = 'test2';
let test2 = 'test2'; //Uncaught SyntaxError: Identifier 'test2' has already been declared 이미 이 식별자는 선언되었다는 에러 발생으로 같은 변수명을 중복으로 쓰는 문제 상황을 예방할수있다.
3. hoisting var 로 선언한 변수는 스코프의 최상단으로 선언부가 끌어올려지게 되고, undefined로 초기화 된다.

console.log(test); // undefined
var test= 1;
console.log(test); //1
